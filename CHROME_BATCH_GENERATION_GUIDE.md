## Efficient Batch Workflow (Key to Speed)

Instead of waiting for each scene individually, use this **parallel batch approach**:

1. **Generate current scene** (click Create)
2. **While it's generating** (first 3-5 seconds), switch focus
3. **Prepare next prompt** (copy from JSON, ready to paste)
4. **When current finishes** (4 variations appear), select best match
5. **Download best variation**
6. **Paste next prompt immediately**
7. **Repeat**

This keeps the pipeline flowing continuously rather than stop-start-stop.
