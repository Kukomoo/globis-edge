## Critical Success Factors

1. **Continuity**: Hawa's headscarf, desk, papers must be identical across all 30 scenes
2. **Grain/Texture**: Every scene needs visible 2–4% stipple (mid-century printing aesthetic)
3. **Camera Flow**: Each scene is a smooth transition from the previous (no jump cuts)
4. **Color Consistency**: Use hex codes (#E8D7A0 sand, #A84C3C rust, #7FAFCA sky, #4A8BA0 teal)
5. **No Text**: Zero subtitles, logos, or labels. Pure illustration

---

## Common Mistakes to Avoid

❌ Generating all 30 at once without reviewing continuity
→ ✅ Batch in groups of 5–7, review before moving on

❌ Allowing Hawa's headscarf to change between scenes
→ ✅ Reference previous scene explicitly in every prompt

❌ Using photorealism instead of illustration
→ ✅ Specify "mid-century editorial style, hand-painted, visible grain"

❌ Device looking too futuristic
→ ✅ "Sand-tone casing, only a single teal indicator light"

❌ Adding text, subtitles, or overlays
→ ✅ Keep it pure visual storytelling

## FAQ

**Q: How long will each scene take?**
A: 3–5 minutes to generate + 1–2 minutes to review = 5–7 min per scene on average.

**Q: What if continuity breaks in Scene 6?**
A: Regenerate Scene 6 with: "Match Scene 5 exactly. Same Hawa, same desk, same palette."

**Q: Can I edit scenes after generation?**
A: Minimal touching up only. Don't manually redraw—it breaks AI aesthetic consistency.

**Q: Should I use 1080p or 4K?**
A: 1080p is sufficient. 4K overkill for this context.

**Q: How many regenerations per scene is normal?**
A: 1–2 average. Some nail it first try; others take 3–4 tries.
