What is Globis Edge?

Globis Edge 2.0 is an offline, on-device caseworker companion for refugee reception centres operating in low-connectivity environments. It turns fragmented identity documents, multilingual testimony, and handwritten notes into structured, audit-safe intake records.

Built for:

  • Caseworkers processing diverse identity documents
  • Interpreters
  • Protection staff
  • Field settings

Core Capabilities

🎯

Tiered Inference

Fast 2B Scout model for translation & light checks. Powerful 4B Analyst for multimodal synthesis.

🎤

Multimodal Intake

Transcribe audio, extract text from documents via OCR, process typed notes—all in one unified pipeline.

⚠️

Conflict Resolution

Detect mismatches across modalities. Never auto-resolve—always human-in-the-loop review.

🛡️

Constitutional Audit

Dual-pass (rule + reasoning) verification. No political affiliation, no ethnic classification, fail-closed design.

🔒

Encrypted Persistence

SQLCipher with value-masked logging. No sensitive data in plain text. Append-only quarantine for audit trail.

📱

Edge Deployment

Runs offline on Raspberry Pi 5 (8GB). True offline-first. No cloud dependency. Deploy locally.
