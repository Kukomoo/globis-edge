## Quality Gates (Don't Compromise)

### Mandatory checks for EVERY scene:
✅ **Visible grain/texture** — 2–4% stipple, like old printmaking  
✅ **Color palette** — warm ochre sand + rust ridges + turquoise sky + teal accents  
✅ **Mid-century aesthetic** — hand-painted look, soft color blocks, not photorealistic  
✅ **Zero text** — no labels, no subtitles, no logos, no watermarks  
✅ **Continuity** — characters/objects don't change appearance across scenes  
✅ **Hawa's scarf** — teal (#4A8BA0), same angle, visible in most scenes  
✅ **Desk/device** — recognizable state evolution, papers organize over time  

### If a scene fails any gate:
1. Regenerate (click Create again with same prompt)
2. Compare all 4 new variations to previous scene
3. Select the one most continuous with previous

## Pro Tips

1. **Batch generation is faster**: Generate 5 scenes, review continuity, move to next batch (vs. one-by-one interruptions)
2. **Keep reference image nearby**: Have your provided art style image open in another window for color/texture comparison
3. **Use color picker**: Verify hex codes of ochre, rust, turquoise, teal in generated images match palette
4. **Screenshot each batch**: Before deleting, screenshot all 4 variations of each scene for backup
5. **Don't regenerate unless necessary**: Most scenes nail the aesthetic on first try; only regenerate if palette is off or continuity breaks
6. **Name consistently**: Use exact format `Scene_[NUM]_[TITLE].png` — this matters for later batch processing
